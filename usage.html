Deadpendency Usage

Installation

Go to the Deadpendency GitHub app and install it. Select which repositories to install the GitHub app on.

Usage

Deadpendency will automatically search for dependency files based on the language GitHub has assigned to your repository.

Refer to language support for information on what dependency files are supported.

Configuration

Configuration is done by including a deadpendency.yaml file in the root directory of your GitHub repository.

Additional Dependency Files

In some cases you have additional, or different dependency files to load than the default ones. These can be loaded like: 

additional-dependency-files:
  - path: other-dependencies.txt
    type: pip-requirements-txt
  - path: ./custom/CustomPipfile
    type: pypa-pipfile

File Types:

  • JavaScript Npm Package File - npm-package-json
  • Python Pip Requirements File - pip-requirements-txt
  • Python Pipenv Pipfile - pipenv-pipfile
  • PHP Packagist Compose File - packagist-composer-json
  • Ruby Bundler Gemfile - bundler-gemfile
  • Ruby Gemspec - rubygems-gemspec
  • Haskell Cabal File - haskell-cabal
  • Haskell Hpack File - hpack-package-yaml

Additional Dependencies

Useful when there are specific additional dependencies you want to be checked that are missing from your dependency files. Or for some reason Deadpendency is not detecting one.

additional-deps:
  javascript:
    - react
    - react-dom
    - name: random-on-github
      repo: some-org/super-random

Ignore Dependencies

Useful when you choose to depend on an unhealthy dependency.

ignore-failures:
  - react-dom
  - angularjs

Configure Deadpendency checks

Each language ecosystem is different. What might be considered unhealthy for a JavaScript dependency may be fine in more niche languages such as Rust.

rules-config:
  no-recent-package-release:
    warn-at-months: 6
    error-at-months: 9
  no-recent-commit:
    warn-at-months: 3
    error-at-months: 6
  few-yearly-commits:
    # warn-at-count: 6 - omitted to use the default
    error-at-count: 3
