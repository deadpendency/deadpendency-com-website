Deadpendency Language Support

Programming Language Supported File Types Default Files Loaded Notes
JavaScript ✅ Npm package.json ./package.json
Python ✅
  • Pip Requirements
  • Pypa Pipfile
  • ./requirements.txt
  • ./Pipfile
 Will not follow requirements.txt includes eg. `-r other-requirements.txt`.
Ruby ✅
  • Gemfile
  • Gemspec
  • ./Gemfile
  • ./**/*.gemspec
 `*.gemspec` search only goes 1 level deep. `./other/dir/mylib.gemspec` will not be found.
PHP ✅ Composer ./composer.json VCS dependencies not yet supported.
Haskell ✅
  • Cabal file
  • Hpack package.yaml
  • ./**/*.cabal
  • ./**/package.yaml
 Only searches 1 directory deep for dependency files.
Java ❌ Maven does not reference source repositories. Cannot be supported.
.NET ❌ Planned
Rust ❌ Planned
Go ❌ Planned
