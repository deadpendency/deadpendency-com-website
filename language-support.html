Deadpendency Language Support

Dependencies are only supported by Deadpendency when hosted on GitHub.

Programming Language Supported File Types Default Files Loaded Registry Deprecation Support Notes
JavaScript ✅ Npm package.json ./package.json ✅
Python ✅
  • Pip Requirements
  • Pipenv Pipfile
  • ./requirements.txt
  • ./Pipfile
 ❌ Will not follow requirements.txt includes eg. -r other-requirements.txt.
Ruby ✅
  • Bundler Gemfile
  • Ruby Gemspec
  • ./Gemfile
  • ./**/*.gemspec
 ❌ *.gemspec search only goes 1 level deep. Git blocks not yet supported.
PHP ✅ Composer file ./composer.json ❌ VCS dependencies not yet supported.
Haskell ✅
  • Cabal file
  • Hpack package.yaml
  • ./**/*.cabal
  • ./**/package.yaml
 ✅ Only searches 1 directory deep for dependency files. Stack and cabal git dependencies not currently supported. Multi package projects require some manual ignoring.
.NET ❌ ✅ Planned
Rust ❌ ❌ Planned
Java ❌ ❌ Maven does not reference source repositories. This impacts all JVM based languages.
Go ❌ ❌ Currently no clear registry API to determine source repository.
Others ❔ If you want a language supported please open an issue.
