Automated dependency health analysis

Get alerted when your software dependencies become inactive, deprecated and more!

How It Works

1
Push a commit to your repo.
2
Deadpendency analyzes your dependencies based on that commit.
3
Recieve a health report via a GitHub check run.

Features

Check dependencies are actively being commited to.
Check for recent package releases.
Check for red flags such as package deprecation or the repository is archived.
Warn when dependencies start to become unhealthy.
Provide an overall report for a snapshot of your dependency project health.
Configure the checks via config.
More checks coming soon!
