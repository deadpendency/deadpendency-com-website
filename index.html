Automated dependency health analysis

A GitHub app that alerts you when your software dependencies become inactive, archived and more!

How It Works

1
Push a commit to your repo.
2
Deadpendency analyzes your dependencies based on that commit.
3
Receive a dependency project health report via a GitHub check run.

Features

Check dependencies are actively being committed to.
Check for recent package releases.
Check for red flags such as package deprecation* or repository archival.
* for registries supporting structured deprecation.
Warn when dependencies start to become unhealthy.
Provide an overall report for a snapshot of your dependencies' project health.
Configure the checks via config.
More checks coming soon!
