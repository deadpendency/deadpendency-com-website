Automated dependency health analysis

Get alerted when your software dependencies become inactive, deprecated and more!

How It Works

  1. Push a commit to your repo.
  2. Deadpendency analyizes your dependencies based on that commit.
  3. Recieve a health report via a GitHub check run.
